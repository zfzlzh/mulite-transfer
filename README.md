# mulite-transfer
可以有多个接收框的穿梭框，elementui风格

### 参数
|  属性名称|  描述 | 类型 | 默认值 |
| --- | --- | --- | --- |
| leftTitle | 左边的标题 | String | '左边的标题' |
| data | 数据源 | Array | [] |
| value | 接收框的数据 | Object | {} |
| rightBoxList | 接收框标题的集合  | Array | ['右边的标题1'] |
